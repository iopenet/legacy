User Module

Overview

This module allows you to manage users, groups, and roles defined in the default security realm.

Users are entities that can be authenticated. Each user is assigned a unique identity within the realm.

To make it easier to administer a large number of users, users can be organized into named groups.

Groups can in turn be assigned membership in other groups.

Only users logged in as a member of the Webmaster group Administrators group can add, delete, or modify a user, group, or role..

Users

Access to your XOOPS Cube web site is controlled through a group-based system by which users are assigned to groups that authorize their access to Modules and blocks content.

Some recent modules can extend User Management providing a role-based system which allows you to assign permissions by User.

You can also extend default User Profile by adding new fields.

Batch Registration

Upload and manage Users using a CSV text file.

Register a batch of members with a CSV file.

Download the User Data CSV file for batch registration.

Open the CVS file and input the member information according to the format - Do not change columns!

When the CSV is successfully registered, the new members will be displayed in User management.

If there is a problem with the content of the CSV file, and it cannot be registered, please refer to the error message to correct the CSV file.
