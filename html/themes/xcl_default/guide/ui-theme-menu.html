Menu

jQuery-UI menu can be created from any valid markup as long as the elements have a strict parent/child relationship. The most commonly used element is the unordered list (ul). Additionally, the contents of each menu item must be wrapped with a block-level DOM element. themeable menu with mouse and keyboard interactions for navigation.
