Selectmenu Widget

Duplicates and extends the functionality of a native HTML select element to overcome the limitations of the native control. The Selectmenu widgets provides a styleable select element replacement. It will act as a proxy back to the original select element, controlling its state for form submission or serialization

Options

The datasource is a native select element. Supports optgroups.

  1. Select a speed, file or number
  2. Custom Rendering, framework icons
  3. Custom images as CSS background.
