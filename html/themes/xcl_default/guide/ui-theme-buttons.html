Buttons

Use buttons and links to trigger actions. Enhance standard form elements inputs and anchors to themeable buttons with appropriate hover and active styles.

Options

  • UI-Button with CSS class
  • UI-Buttons with Icons
  • Widget UI-Buttons with Icons
  • Style with custom classes
  • SVG icons styled with CSS

    • Tag "i"with class i-[actioname]

Button with SVG icons styled using CSS

The tag <i> with a class "i-[actioname]" renders an inline SVG icon. This avoids the JavaScript overhead if you don't need any of the methods provided by the button widget.



<button class="ui-button" title="i-add">
    <i class="i-add"></i> i-add
</button>
<button class="ui-button" title="i-alert">
    <i class="i-alert"></i> i-alert
</button>
<button class="ui-button" title="i-delete">
    <i class="i-delete"></i> i-delete
</button>
<button class="ui-button" title="i-edit">
    <i class="i-edit"></i> i-edit
</button>
