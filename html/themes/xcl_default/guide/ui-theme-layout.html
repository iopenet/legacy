XC—Layout helper based on CSS flexbox specification using custom html attributes — layout and self — extended with CSS custom properties and scalable spacing attributes, margin and padding. It can also be used as a fallback for a responsive layout in case a theme or template style cannot be rendered.

Options

  1. CSS reset that provides better cross-browser consistency in the default styling of HTML elements.
  2. CSS box-sizing Property to all elements includes the 'padding' & 'border' in elements total width & height.
  3. Layout helper based on CSS flexbox specification designed to serve
  4. quick flexbox shorthand by using custom HTML attributes
  5. Position : layout and self (parent and child elements)
  6. Spacing : p** and m** for padding and margin, -n** negative
  7. native CSS feature "scroll-behavior:smooth"
