Badges

Badges are labels which hold small amounts of information. This component is composed of text or icon inside a span element. Based on X-Layout helper, Badges are designed to serve a quick shorthand by using custom HTML data attributes.

Options

Badges can also be used inside button elements.

  • CSS custom properties
  • Before and after text/value
  • Distinct colors
  • Position [left, top, right, bottom]
  • Shape [badge, circle, dot, pill]
