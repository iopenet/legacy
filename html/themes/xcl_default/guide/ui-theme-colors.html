Color scheme Dark mode

The color scheme is an association of colors based on an organizational system.
Basically, it's a set of colors that work well together to create a unified aesthetic and consistency in User Interface design.

Options

X-Layout CSS Custom properties

The color scheme is set with CSS custom properties (variables).
The calc() function automatically generates distinct colors from values set in "_ui_root.css".

Components that can be reused anywhere and styled with their own [data-theme="color-scheme-indigo"] attribute.

HSL and HSV are alternative representations of the RGB color model, designed in the 1970s by computer graphics researchers to more closely align with the way human vision perceives color-making attributes.
